# HowMuchIsLeft
A mod for My Summer Car that shows the content of some items.

## Dependencies
- [MSCLoader](https://github.com/piotrulos/MSCModLoader/releases)

## Supported Items
- Two Stroke Fuel
- Motor Oil
- Brake Fluid
- Coolant
