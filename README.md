# HowMuchIsLeft
A mod for My Summer Car that shows the content of some items when you look at them. See currently supported items below.

## Dependencies
- [MSCLoader](https://github.com/piotrulos/MSCModLoader/releases)

## Supported Items
- Two Stroke Fuel
- Motor Oil
- Brake Fluid
- Coolant
- Ground Coffee
- Grill Charcoal
- Spray Can
- Mosquito Spray
- Fuse Package
- R20 Battery Box
- Spark Plug Box
- Teimo's Advert Pile
- Fire Extinguisher
